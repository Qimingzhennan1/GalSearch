# Gal Search MVP

**便携式 Windows 桌面搜索工具** — 搜索和浏览 gal 资源分享页面。
拷贝整个文件夹到任意 Windows 电脑即可使用，无需安装，无额外依赖。

---

## 📦 文件清单

| 文件 | 说明 |
|------|------|
| `GalSearch.ps1` | 主程序（PowerShell WinForms） |
| `GalSearch.vbs` | **✅ 静默启动器** — 无控制台窗口，双击即用（推荐） |
| `Run-GalSearch.bat` | 备用启动器 — 会显示控制台窗口（用于调试） |
| `install.bat` | 双击安装桌面快捷方式 |
| `install.ps1` | 快捷方式安装脚本（被 `install.bat` 调用） |
| `README.md` | 本说明文档 |

---

## 🚀 使用方法

### 方式一：直接运行（推荐）

双击 **`GalSearch.vbs`** — 无控制台窗口，像真正的桌面应用一样启动。

### 方式二：安装快捷方式

双击 **`install.bat`**，会在桌面创建快捷方式，之后双击桌面图标启动。

### 方式三：控制台模式（调试用）

双击 `Run-GalSearch.bat` 启动，控制台窗口保持打开以便查看错误输出。

## ✨ 功能

- **本地缓存** — 搜索结果自动存储在 `%LOCALAPPDATA%\GalSearchMVP\index.json`
- **关键词搜索** — 按 Enter 搜索，匹配标题/摘要/URL/标签
- **在线搜索** — 实时抓取百度搜索结果，自动缓存到本地
- **来源筛选** — 按 All / Baidu 过滤
- **排序** — 按相关性 / 时间 / 标题
- **详情面板** — 选中结果后右侧显示标题、来源 URL、缓存时间、标签、摘要
- **标签编辑** — 可编辑标签（逗号分隔），点击 Save 持久化保存
- **打开链接** — 双击结果行，或点击"Open source link"按钮
- **复制 URL** — 一键复制分享链接到剪贴板

## ⌨️ 快捷键

| 快捷键 | 动作 |
|--------|------|
| `Enter` | 搜索 |
| `Ctrl+F` / `Ctrl+L` | 聚焦搜索框 |
| `Ctrl+R` | 重新加载缓存 |
| `Ctrl+C` | 复制选中项 URL |
| `Esc` | 清空搜索框 / 取消选中行 |

## 📋 菜单栏

| **查看** | 排序: 相关性 / 时间 / 标题 |

## 📦 便携性说明

- 拷贝整个文件夹到 **任意 Windows 电脑**（U 盘、移动硬盘、网络共享均可）
- 无需安装 .NET、Node.js、Python 等运行时
- 所有数据存储在 `%LOCALAPPDATA%\GalSearchMVP\`，卸载只需删除文件夹
- 运行环境：Windows 7+，PowerShell 5.1+（Windows 自带）

---

## ⚠️ 法律边界

- **只做搜索和导航** — 不实现下载自动化
- **不做登录处理** — 不处理需要登录才能访问的资源
- **不绕过访问限制** — 严格遵守网站规则
- **百度解析依赖 HTML 结构** — 百度页面改版时解析可能失效
